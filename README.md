# Autism Detection System 🧠

## Setup

### 1. Clone the repo

git clone https://github.com/yassmen1/GradProject.git

### 2. Open folder

cd GradProject

### 3. Create virtual environment

python -m venv venv
venv\Scripts\activate

### 4. Install dependencies

pip install -r requirements.txt

### 5. Add .env file

Create .env and add your keys

### 6. Run project

python app.py
